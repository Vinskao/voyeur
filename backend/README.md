## 安裝

### 使用Poetry安裝
確保您已經安裝了[Poetry](https://python-poetry.org/)，然後運行以下命令來安裝項目依賴：

```bash
pip install poetry
poetry install
python3 -m voyeur.connect_metrics
python3 manage.py runserver
poetry run python3 manage.py runserver
poetry run python manage.py runserver
```
